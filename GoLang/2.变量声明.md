## 变量声明的三种方式

- ##### var 变量名 变量类型

    ```go
    var i int
    var i, j int
    ```

- ##### var 变量名 = 变量值   => 启用类型推到，根据变量值来推到变量的类型

    ```go
    var i = 10
    var i, j = 10, "str"
    ```

- ##### 变量名 := 变量值      => 启用类型推到，根据变量值来推到变量的类型

    ```go
    i := 10
    i, j := 110, "str"
    ```



## 在全局作用域下一次性声明多个变量：使用`var()`的形式

```go
var (
    x = 50
    y = "sdf"
)
```
